# 🎬 IMDB Review Sentiment Analysis (ML)

This project builds a Machine Learning model to classify IMDB movie reviews as **positive** or **negative** using NLP techniques.

---

## 🚀 Features

* Text preprocessing (tokenization)
* Word embeddings using Word2Vec
* Document vector creation (average embeddings)
* ML model training (classification)

---

## 🛠️ Tech Stack

* Python
* NumPy, Pandas
* NLTK
* Gensim (Word2Vec)
* Scikit-learn

---

## 📊 Workflow

1. Load dataset
2. Clean & tokenize text
3. Train Word2Vec model
4. Convert reviews → vectors
5. Train ML model
6. Evaluate performance

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
```

Run the notebook:

```bash
jupyter notebook
```

---

## 📌 Result

The model predicts sentiment of movie reviews using word embeddings and ML algorithms.
